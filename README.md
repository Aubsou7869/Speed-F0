SPEED-FO Sport Motorcycle Management System

Project Description
The Sport Motorcycle Management System is a console-based Java application designed to manage information about sport motorcycles. The application allows users to add, view, update, and delete motorcycle records using a terminal interface. The system uses object-oriented programming concepts such as abstraction, inheritance, interfaces, composition, and polymorphism. SQLite is used as the database for storing motorcycle information permanently.

Project Tasks
Task 1: Set up the development environment
Install Java Development Kit (JDK)
Configure Eclipse or IntelliJ IDEA
Set up Git and GitHub repository
Add SQLite JDBC library (sqlite-jdbc-3.36.0.1.jar)

Task 2: Design the application
Plan application structure and class hierarchy
Design database table structure
Create UML-style class definitions

Task 3: Develop application classes
Create abstract Motorcycle class
Create SportMotorcycle subclass
Create Engine composition class
Create DatabaseOperations interface
Create MotorcycleDB database management class
Create MainApp terminal interface class

Task 4: Implement object-oriented programming concepts
Use inheritance between Motorcycle and SportMotorcycle
Use abstraction with the Motorcycle abstract class
Use polymorphism through method overriding
Use composition with the Engine class
Use interfaces for CRUD operations

Task 5: Implement CRUD functionality
Create motorcycle records
Read/display motorcycle records
Update motorcycle records
Delete motorcycle records

Task 6: Connect to SQLite database
Create SQLite database connection
Create motorcycles table
Implement SQL queries using JDBC
Store and retrieve motorcycle data

Task 7: Test the application
Test user input and output
Test CRUD database operations
Debug syntax and runtime errors

Task 8: Document the project
Create README documentation
Add in-code comments and class documentation
Prepare project proposal and design documents
